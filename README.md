# safari-firefox
A highly refined, premium CSS theme that brings the sleek, flawless macOS Safari aesthetic to Firefox and Firefox based browsers. Built on top of the WhiteSur theme, this project fixes common visual glitches, perfects the UI elements, and introduces true Safari-like "Compact Mode" to firefox

SAFARI-FIREFOX

Instructions:

    This theme is compatible with the latest release of Firefox and works on windows and linux
    To enable adaptive colors, you need to install the Adaptive Tab Bar Color extension. If not, the tabs will appear transparent.

    <img width="1920" height="1200" alt="image" src="https://github.com/user-attachments/assets/254d8b31-c13b-49c8-b86e-588d97ffdbc6" />
 
  Installation

    Download the theme file and unzip the chrome folder into your profile folder.

    Configuration

    about:config
        Set toolkit.legacyUserProfileCustomizations.stylesheets to true.
        Set svg.context-properties.content.enabled to true.
        Set widget.non-native-theme.use-theme-accent to true.

    Required settings
        Move all toolbar buttons to the top, and the tab bar does not display any buttons.
        If extension (Adaptive Tab Bar Color) is enabled, set all colors in the Options (theme builder) to 0%.
